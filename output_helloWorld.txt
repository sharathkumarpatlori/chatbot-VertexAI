==== Description of Block 1 ====
The line `#include <iostream>` includes the iostream standard library.
 This library provides input/output functionalities, specifically allowing your program to interact with the console (like printing text to the screen).

`int main() { ...
}` is the main function where program execution begins.
 Everything inside the curly braces `{}` will be executed.

`std::cout << "Hello, World!" << std::endl;` prints the text "Hello, World!" to the console.
 `std::cout` is the standard output stream (usually your screen), `<<` is the insertion operator sending data to the stream, and `std::endl` inserts a newline character, moving the cursor to the next line.

`return 0;` indicates that the program executed successfully.
 A non-zero return value usually signifies an error.
